
## 5. Current 3D Topology Canvas — Professional-Grade Improvements

### 5.1 Rendering Quality

| # | Improvement | Current State | Professional Standard |
|---|-------------|---------------|----------------------|
| 5.1.1 | **Anti-aliasing** | WebGL anti-aliasing enabled but no MSAA/FXAA post-processing | Add `EffectComposer` with FXAA or SMAA pass for smoother edges. |
| 5.1.2 | **Shadows** | No shadows | Add soft shadows from directional light. `renderer.shadowMap.enabled = true`, `PCFSoftShadowMap`. Ground plane receives shadows. |
| 5.1.3 | **Ambient Occlusion** | None | Add SSAO (Screen Space Ambient Occlusion) post-processing for depth perception in dense piping. |
| 5.1.4 | **Material quality** | Basic `MeshStandardMaterial` with flat colors | Add PBR materials: metallic pipes (metalness: 0.4, roughness: 0.5), matte flanges, glossy valve bodies. Optional: environment map for reflections. |
| 5.1.5 | **Edge rendering** | No edges visible on geometry | Add outline/edge detection shader for CAD-like wireframe overlay on solid geometry. Toggle with `E` key. |
| 5.1.6 | **Background** | Solid white (`#ffffff`) | Professional gradient background (light: `#f0f2f5` to `#dde1e7`, dark: `#1a1f2e` to `#0d1117`). Toggle dark/light theme. |

### 5.6 Performance

| # | Improvement | Current State | Professional Standard |
|---|-------------|---------------|----------------------|
| 5.6.1 | **Level of Detail (LOD)** | All components rendered at full detail regardless of distance | Implement LOD: far components rendered as simple lines, medium as low-poly cylinders (8 segments), close as full detail (16 segments). |
| 5.6.2 | **Instanced rendering** | Each component creates its own geometry | Use `InstancedMesh` for components of the same type and bore — single draw call for all pipes, single for all bends, etc. |
| 5.6.3 | **Frustum culling** | Three.js default frustum culling | Already handled by Three.js, but ensure bounding spheres are computed for all meshes. |
| 5.6.4 | **Geometry pooling** | New geometry created per component | Create geometry pools per type+bore combination. Reuse cylinder geometries across all pipes of the same bore. |
| 5.6.5 | **Progressive rendering** | All components rendered on first frame | For large models (1000+ components), render in batches of 50 per frame to avoid frame drops during initial load. |
